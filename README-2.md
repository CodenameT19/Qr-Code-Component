### What I learned


This project helped me practice layout basics, especially:

### ✔ Centering the card on the page

I learned how to use flexbox to center the component on the screen:

```css
body {
  min-height: 100vh;
  display: flex;
  align-items: center;
  justify-content: center;
  background-color: hsl(212, 45%, 89%);
}

### I used a container that shrinks to fit the QR code and text:

.design {
  background-color: white;
  display: flex;
  justify-content: center;
  align-items: center;
  flex-direction: column;
  border-radius: 10px;
}

### I learned to use margin: 1rem to create consistent spacing: 

.attribution {
  text-align: center;
  margin-top: 1rem;
}
